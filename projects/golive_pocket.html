Core Screens

  • Compliance Disclaimer — first-launch mandatory acceptance screen
  • Project List (Home) — active projects sorted by next go-live date
  • Project Detail — calendar + list views with quick action buttons
  • Go-Live Form — create/edit releases with Red Advisory warnings
  • Checklist View — readiness checklist with progress tracking
  • On-Call Manager — schedule management with conflict warnings
  • Vacation Manager — time-off tracking with overlap detection
  • Red Advisory Manager — high-risk date marking
  • Reminder Manager — personal reminders linked to go-live
  • Project Settings — update project properties

Key Features Implemented

Project-first workflow — must create project before adding entries

Risk color coding — Green / Amber / Red

Phase tracking — DEV / UAT / PROD

Lifecycle status — Active / Completed / On Hold / Cancelled

Days-left badges — D-x, Today, Overdue

Automatic red override — overdue go-lives show as red

Conflict detection — vacation vs on-call + red advisory warnings

Compliance pattern warnings — URLs, ticket IDs, system names

Calendar overlay view — visual planning for releases

Readiness checklist — custom items + progress tracking

LocalStorage persistence — app state retained in browser
